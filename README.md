<h1 align="center">Auto Composer</h1>

<p align="center">Procedural MIDI composition from existing scores.</p>

## Overview

A cross-platform CLI tool that analyzes MIDI files and generates new musical compositions procedurally, inspired by the patterns, harmonies, and structures found in the input score.
